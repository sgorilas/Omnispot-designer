Prerequisites

The Omni-Spot Designer is a standalone application. As long as a Java runtime environment is available it should function properly as far as the designing of Omni-Spot deployment descriptors and their export for usage by the rest of the system.

In order for the deployment preview to work correctly, the VLC player must be installed on the machine and the path to directory where VLC is installed must be added to the PATH environment variable. Both these actions are described below.

Installing the VLC player

You can find the latest VLC player installer here.

Adding to the PATH environment variable

After you have installed the VLC player, you must add the path to directory where VLC is installed to the PATH environment variable. To do this bring up the system properties page for the computer (either right clicking on the computer icon and selecting properties, or from the control panel). Select the Advanced tab in the System Properties window. This should look like this:

Click on the Environment Variable button. The environment variables editor should show up:

On the System Variables section select the Path system variable and click on the Edit button below. It will bring an editor for changing the value of the Path environment variable. At the end of the variable's value append a semicolon followed by the path to VLC installation directory (usually C:\Program Files\VideoLAN\VLC). Click OK on all the editors and dialogs to make your changes take effect.
