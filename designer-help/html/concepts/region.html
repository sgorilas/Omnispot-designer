Region

A region is a root container of components. Typically a region will contain a single component (such as a video component).

A region has a set of properties that can be modified in the property sheet view. These are:

  • Name: A name for the region. This is only used in the designer and in error reporting.
  • Transparent: A boolean that specifies if the region is to allow its contained components to have an invisible background. This requires the cooperation of the contained components. Currently only the ticker component supports this functionality.
  • Background Color: Specifies the background color of the region. This will only show through if the region is not fully covers by contained components.
  • Location: The location of the region.
  • Size: The size of the region.

A region cannot overstep the boundaries of the deployment for which it is defined.
